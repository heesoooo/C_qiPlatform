## 1. Node.js 설치
gulp를 사용하기 위해서 node.js 를 다운로드 받기

* https://nodejs.org      ```


node와 npm이 제대로 설치되었는지 확인

 ```
* [ 노드 버전 확인 ]   node -v 
* [ npm 버전 확인 ]   npm -v
 ```


## 2. npm 의존 설치

```
npm install
```

## 3.package 실행

```
npm start
```
